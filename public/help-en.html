Novel Master v1.0 - AI-Powered Novel Writing Assistant

Overview

Novel Master is an AI-powered novel writing assistant that supports setting management, story writing, character analysis, plot recording, and visual map features.

Quick Start

1. Enter your DeepSeek API key in Settings → 2. Create world settings in Setting mode → 3. Switch to Story mode to start writing → 4. Use Record to update plot summary

Dual Mode Writing

Setting mode for world-building, Story mode for narrative writing.

Smart Memory

AI automatically remembers plot, characters, and important events.

Visual Maps

Interactive map editor for scenes and story flow.

Branch Management

Support for story branches - explore different plotlines.

Sidebar Features

The sidebar provides session management and basic settings.

New Story

Create a new writing session in Setting mode.

Import

Import session backups from JSON files.

Backup

Export sessions as JSON files.

Export Novel

Export story content as Markdown format.

Session Management

  • Click session name to switch sessions
  • Double-click or click edit icon to rename
  • Click delete icon to remove session

Writing Modes

The mode toggle at the top allows switching between Setting and Story modes.

Setting Mode

For creating fundamental story settings.

  • Create world settings and outlines
  • Design character profiles
  • Define story background and rules
Story Mode

For writing the main narrative content.

  • Write story narrative
  • Dialogue and scene description
  • Plot development

Tip

Click the "Add to Setting" button (BookPlus icon) next to a message to add it to world settings. It will automatically switch to Story mode afterward.

Message Actions

Hover over messages to reveal the action toolbar.

Branch Navigation

When a message has branches, a navigator (e.g., 1/3) appears. Click arrows to switch between branches. You can keep multiple plotlines simultaneously.

Input Tips

  • Type / to reference bookmarked content
  • Enter to send, Shift+Enter for new line
  • On mobile, Enter creates new line by default

Canvas Mode

Canvas mode provides a fullscreen immersive message editing and viewing experience. Click the expand icon next to a message to enter.

Main Features

Editing Operations

  • Ctrl/Cmd + S: Save edited content
  • Ctrl/Cmd + Enter: Save and send to AI
  • Esc: Cancel editing, close canvas
  • Tab: Switch focus between paragraphs

Use Cases

Canvas mode is especially useful when editing longer content, such as modifying AI-generated story outlines, adjusting chapter structure, or polishing dialogue.

Map Visualization

Interactive map editor for visualizing story scenes and character relationships. Create worlds, regions, and nodes to form a complete story world.

Canvas Operations

  • Right-click canvas: Open creation menu to create new nodes, regions, or worlds
  • Drag node: Move node to any position
  • Connect ports: Drag from one node's edge port to another to establish connection
  • Scroll wheel: Zoom in or out
  • Middle-click / Alt+drag: Pan the entire view
  • Delete key: Delete selected nodes or connections
  • Click empty area: Deselect

Z-Axis Connections

Triangle ports at the top of nodes are for vertical connections (like stairs, elevators), representing floor transitions. Dashed lines indicate vague or uncertain connections.

Developer Mode

Enabling Developer Mode allows manual map instruction injection, directly creating and modifying map structures through natural language descriptions.
