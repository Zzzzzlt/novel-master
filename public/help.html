小说大师 v1.0 - AI 辅助小说创作工具

概述

小说大师是一款基于 AI 的小说创作辅助工具，支持设定管理、故事写作、角色分析、情节记录和可视化地图等功能。

快速开始

1. 在设置中输入 DeepSeek API 密钥 → 2. 在设定模式中创建世界观和角色 → 3. 切换到故事模式开始写作 → 4. 使用记录功能更新情节概要

双模式创作

设定模式用于构建世界观，故事模式用于写作正文。

智能记忆

AI 自动记忆情节概要、角色设定和重要事件。

可视化地图

交互式地图编辑器，展示场景和故事脉络。

分支管理

支持故事分支，可在不同剧情线之间切换。

侧边栏功能

侧边栏提供会话管理和基础设置功能。

新建故事

创建新的写作会话，自动切换到设定模式。

导入

从 JSON 文件导入会话备份。

备份

导出会话为 JSON 文件。

导出小说

将故事正文导出为 Markdown 格式。

会话管理

  • 点击会话名称切换到该会话
  • 双击或点击编辑图标重命名会话
  • 点击删除图标删除会话

创作模式

顶部模式切换按钮允许在设定模式和故事模式之间切换。

设定模式

用于创建小说的基础设定。

  • 创建世界观设定和大纲
  • 设计角色设定和关系
  • 定义故事背景和规则
故事模式

用于写作小说正文内容。

  • 写作故事正文
  • 对话和场景描写
  • 情节发展和推进

提示

点击消息旁的「添加到设定」按钮（BookPlus 图标），可将内容添加到世界观设定。添加后会自动切换到故事模式。

消息操作

鼠标悬停到消息上会显示操作按钮栏。

图标 功能 说明
复制 复制消息内容到剪贴板
收藏 标记重要章节
重新生成 重新请求 AI 生成
编辑 修改消息内容
画布模式 全屏展开查看
分支 创建新的故事分支
添加到设定 仅设定模式可见，将内容加入世界观设定
分析记录 仅故事模式可见，分析并记录到记忆
删除 删除该消息

分支导航

当消息存在分支时，会显示分支导航器（如 1/3），点击箭头在分支间切换。可以同时保留多条剧情线。

输入框技巧

  • 输入 / 引用书签内容
  • Enter 发送消息，Shift+Enter 换行
  • 手机端默认 Enter 换行

画布模式

画布模式提供全屏沉浸式的消息编辑和查看体验。点击消息旁的展开图标即可进入。

主要功能

编辑消息内容
添加新段落
字数统计
Markdown 支持

编辑操作

  • Ctrl/Cmd + S：保存编辑内容
  • Ctrl/Cmd + Enter：保存并发送给 AI
  • Esc：取消编辑，关闭画布
  • Tab：在段落间切换焦点

使用场景

画布模式特别适合需要编辑较长内容时使用，例如修改 AI 生成的故事大纲、调整章节结构或润色对话内容。

右侧面板

右侧面板提供上下文记忆、AI 建议和地图功能，通过顶部标签页切换。

上下文标签页

记忆配置

记忆轮数控制 AI 参考的历史消息数量。开启快速创作后，生成内容会自动记录分析。

AI 建议

输入指导内容获取剧情建议，点击建议可插入到输入框。

情节概要

记录故事主线和重要事件。

角色列表

以卡片形式展示角色设定。

长期记忆

记录重要细节和伏笔。

世界观设定

展示所有设定内容。

地图标签页

神经网络视图

故事地图的缩略预览，点击「打开地图界面」进入全屏编辑模式。

自动同步：开启后 AI 生成的内容会自动更新地图。
从故事更新：手动触发 AI 分析并更新地图。

地图可视化

交互式地图编辑器，用于可视化故事场景和角色关系。可以创建世界、区域和节点，形成完整的故事世界。

层级结构

世界 (World) 区域 (Region) 节点 (Node)

节点类型

房间 地标 传送点 未知

连接类型

道路 门 传送门 楼梯

画布操作

  • 右键画布：打开创建菜单，可创建新节点、区域或世界
  • 拖拽节点：移动节点到任意位置
  • 连接端口：从节点的边缘端口拖拽到另一个节点建立连接
  • 滚轮缩放：放大或缩小视图
  • 中键/Alt+拖拽：平移整个视图
  • Delete 键：删除选中的节点或连接
  • 点击空白处：取消选择

Z 轴连接

节点顶部的三角形端口用于垂直连接（如楼梯、电梯），表示楼层切换。虚线表示模糊或不确定的连接关系。

开发者模式

开启开发者模式后，可以手动注入地图指令，直接通过自然语言描述来创建和修改地图结构。
